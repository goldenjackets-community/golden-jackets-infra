# Post-Setup Checklist

After running `./setup-chapter.sh`, complete these manual steps:

## Immediate (before announcing)

- [ ] Upload chapter leader photo → `assets/members/{name}.jpg` (push to repo)
- [ ] Upload jacket image → `assets/jacket-{code}.png` (push to repo)
- [ ] Customize region/state dropdowns in `index.html` (stateFilter + applyState selects)
- [ ] Highlight country on global map (`~/golden-jackets-global/index.html`)
  - Open site, inspect the country path, find the line number
  - Change `fill="#1a1a35"` → `fill="#b8860b"` + add `class="active-country" style="cursor:pointer;transition:fill 0.3s"` + `stroke="none"`
  - ⚠️ Some countries have multiple paths — check visually
- [ ] Adjust map pin position in `index.html` (`top:X%;left:Y%`) to center on leader's city

## After DNS propagates (1-48h)

- [ ] ACM certificate validates automatically (check in console)
- [ ] Add custom domain to CloudFront:
  ```bash
  aws cloudfront get-distribution-config --id {CF_ID} --profile gj > /tmp/cf.json
  # Edit: add Aliases + ViewerCertificate with ACM ARN
  aws cloudfront update-distribution --id {CF_ID} --profile gj --cli-input-json file:///tmp/cf.json
  ```
- [ ] Create Route53 A + AAAA alias records:
  ```bash
  aws route53 change-resource-record-sets --hosted-zone-id {HZ_ID} --profile gj --change-batch '{
    "Changes": [
      {"Action":"CREATE","ResourceRecordSet":{"Name":"{DOMAIN}","Type":"A","AliasTarget":{"HostedZoneId":"Z2FDTNDATAQYW2","DNSName":"{CF_DOMAIN}","EvaluateTargetHealth":false}}},
      {"Action":"CREATE","ResourceRecordSet":{"Name":"{DOMAIN}","Type":"AAAA","AliasTarget":{"HostedZoneId":"Z2FDTNDATAQYW2","DNSName":"{CF_DOMAIN}","EvaluateTargetHealth":false}}},
      {"Action":"CREATE","ResourceRecordSet":{"Name":"www.{DOMAIN}","Type":"A","AliasTarget":{"HostedZoneId":"Z2FDTNDATAQYW2","DNSName":"{CF_DOMAIN}","EvaluateTargetHealth":false}}},
      {"Action":"CREATE","ResourceRecordSet":{"Name":"www.{DOMAIN}","Type":"AAAA","AliasTarget":{"HostedZoneId":"Z2FDTNDATAQYW2","DNSName":"{CF_DOMAIN}","EvaluateTargetHealth":false}}}
    ]
  }'
  ```

## After site is live

- [ ] Test: `curl -sI https://{DOMAIN}` → should return 200
- [ ] Send Chapter Leader Guide PDF
- [ ] Announce on LinkedIn / WhatsApp Chapter Leaders group
- [ ] Update global site stats if needed (`data.json`)
